## 📋 项目信息

- **产品名称**: DB-Doctor
- **产品定位**: 基于 DeepSeek/LLM 的 MySQL 数据库"自动驾驶"诊疗平台
- **核心受众**: Java 开发、DBA、架构师，以及对 DeepSeek/本地大模型感兴趣的技术爱好者
- **推广周期**: 建议前两周为冲刺期，后续持续迭代和运营

## 💎 核心营销话术（卖点提炼）

在所有宣传中，持续强调以下几点：

1. **数据不出域**: 支持 Ollama + 本地模型，绝对安全，企业级应用首选
2. **非侵入**: 无需改业务代码，无需引入 Jar 包，独立部署，对现有系统零影响
3. **DeepSeek R1 加持**: 借势顶流 AI 模型，强调高精准诊断能力
4. **多 Agent 协作**: 突出技术深度，拟人化设计，解决复杂问题
5. **开箱即用**: Docker 一键部署，5分钟上手
6. **解放 DBA/开发**: 从繁琐的慢查询诊断中解脱，提升效率
