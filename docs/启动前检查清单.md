## ⚠️ 常见启动问题

### 问题 1: H2 数据库连接失败

**错误信息**:
```
Connection is broken: "java.net.ConnectException: Connection refused"
```

**解决方法**:
- 检查 `data` 目录是否存在
- 确保没有其他进程占用H2数据库端口
- 检查 `application.yml` 中的数据库连接配置

### 问题 2: 表创建失败

**错误信息**:
```
Table "notification_queue" not found
```

**解决方法**:
1. 检查 `spring.jpa.hibernate.ddl-auto` 是否为 `update`
2. 检查 `spring.jpa.database-platform` 是否为 `org.hibernate.dialect.H2Dialect`
3. 检查实体类是否有 `@Entity` 注解

### 问题 3: 定时任务不执行

**可能原因**:
- 没有待发送的通知记录（正常情况）
- Cron 表达式配置错误
- Spring `@EnableScheduling` 未启用

**验证方法**:
- 检查日志中是否有 `@Scheduled` 相关的错误
- 手动触发慢查询分析，产生待发送的通知
- 检查 `application.yml` 中的 `db-doctor.notify.batch-cron` 配置
