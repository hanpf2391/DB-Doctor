# DB-Doctor 快速开始指南

## 📦 下载安装

### 方式一：直接下载（推荐）

1. 访问 [GitHub Releases](https://github.com/hanpf2391/DB-Doctor/releases)
2. 下载最新版本的发布包：
   - **Windows 用户**: `DB-Doctor-v3.x.x.zip`
   - **Linux/Mac 用户**: `DB-Doctor-v3.x.x.tar.gz`
3. 解压到本地目录

### 方式二：从源码构建

```bash
git clone https://github.com/hanpf2391/DB-Doctor.git
cd DB-Doctor
./scripts/package-release.sh
```

## ⚙️ 系统要求

| 项目 | 要求 |

## 🚀 快速启动（3 步）

### Windows 用户

#### 步骤 1: 检查 Java 环境

打开命令提示符（CMD），输入：

```cmd
java -version
```

应显示类似：`java version "17.x.x"`

> **提示**: 如果提示"不是内部或外部命令"，请先安装 [Java 17](https://www.oracle.com/java/technologies/downloads/)

#### 步骤 2: 启动应用

双击运行启动脚本：

```
scripts\start.bat
```

或命令行运行：

```cmd
cd DB-Doctor-v3.x.x
scripts\start.bat
```

#### 步骤 3: 访问系统

浏览器访问：**http://localhost:8080**

默认账号：
- 用户名: `dbdoctor`
- 密码: `dbdoctor`

---

### Linux/Mac 用户

#### 步骤 1: 检查 Java 环境

```bash
java -version
```

应显示类似：`openjdk version "17.x.x"`

> **提示**: 如果未安装，执行：
> ```bash
> # Ubuntu/Debian
> sudo apt install openjdk-17-jre
>
> # CentOS/RHEL
> sudo yum install java-17-openjdk
>
> # macOS
> brew install openjdk@17
> ```

#### 步骤 2: 启动应用

```bash
# 进入目录
cd DB-Doctor-v3.x.x

# 赋予执行权限
chmod +x scripts/*.sh

# 启动应用
./scripts/start.sh
```

#### 步骤 3: 访问系统

浏览器访问：**http://localhost:8080**

默认账号：
- 用户名: `dbdoctor`
- 密码: `dbdoctor`

## ⚙️ 配置说明

### 配置文件位置

```
DB-Doctor-v3.x.x/
└── config/
    └── application-local.yml          # 主配置文件（首次启动自动创建）
```

### 必填配置项

首次运行时，应用会自动从模板创建配置文件。你需要编辑 `config/application-local.yml`：

#### 1. 目标数据库配置（必填）

```yaml
db-doctor:
  target-db:
    url: jdbc:mysql://your-mysql-host:3306/information_schema
    username: your_mysql_username
    password: your_mysql_password
```

#### 2. AI 服务配置（可选）

如果启用 AI 智能分析：

```yaml
langchain4j:
  open-ai:
    api-key: your-api-key-here
    base-url: https://api.openai.com/v1
```

#### 3. 邮件通知配置（可选）

如果需要邮件通知：

```yaml
db-doctor:
  notify:
    email:
      enabled: true
    mail:
      smtp:
        host: smtp.qq.com
        port: 587
        username: your-email@qq.com
        password: your-smtp-password
```

## 🔧 常见问题

### Q1: 提示"未找到 Java"或"java: command not found"

**原因**: 未安装 Java 或 Java 版本过低

**解决**: 安装 Java 17 或更高版本

- Windows: [Oracle Java 下载](https://www.oracle.com/java/technologies/downloads/)
- Linux: `sudo apt install openjdk-17-jre`
- macOS: `brew install openjdk@17`

---

### Q2: 端口 8080 被占用

**原因**: 默认端口 8080 已被其他程序使用

**解决**: 修改 `config/application.yml` 中的端口配置：

```yaml
server:
  port: 9090  # 改为其他端口
```

---

### Q3: 无法连接到目标 MySQL 数据库

**原因**: 数据库连接配置错误或网络不通

**解决**:
1. 检查 `config/application-local.yml` 中的数据库配置
2. 确保数据库服务器可访问（ping 测试）
3. 确认数据库用户有足够权限

---

### Q4: 如何查看日志

**位置**: `logs/db-doctor.log`

**实时查看**（Linux/Mac）:
```bash
tail -f logs/db-doctor.log
```

---

### Q5: 如何停止应用

**Windows**: 按 `Ctrl + C` 或直接关闭命令行窗口

**Linux/Mac**: 按 `Ctrl + C` 或执行 `kill $(pgrep -f db-doctor.jar)`

---

### Q6: 首次登录后密码修改

首次登录后强烈建议修改默认密码：

1. 登录系统
2. 点击右上角用户菜单
3. 选择"修改密码和用户名"
4. 设置新密码

---

### Q7: 数据保存在哪里？

**内置 H2 数据库**: `data/db-doctor-internal.mv.db`

包含以下数据：
- 慢查询指纹
- AI 分析报告
- 通知历史记录
- 系统配置

**备份建议**: 定期备份 `data/` 目录
